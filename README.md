# Proyecto-final_PDP

este es el repositorio del proyecto final de la asignatura de Paradigmas de Programación de la universidad Pontificia Bolivariana. 

## Descripción del Proyecto
El proyecto consiste en una aplicación cuyo objetivo es brindar a los usuarios una mala experiencia de usuario intencionalmente. La aplicación está diseñada para ser frustrante y difícil de usar, con el fin de destacar la importancia de una buena experiencia de usuario en el desarrollo de software.

## Tecnologías Utilizadas
- C# 
- .NET Framework
- Windows Forms
- Visual Studio
## Instalación
1. Clona este repositorio en tu máquina local.
2. Abre el proyecto en Visual Studio.
3. Restaura las dependencias y paquetes necesarios.
4. Compila y ejecuta la aplicación.
## Uso
1. Navega por la aplicación y experimenta la mala experiencia de usuario.
2. Proporciona retroalimentación sobre cómo mejorar la experiencia de usuario.
